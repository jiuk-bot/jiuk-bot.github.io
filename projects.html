직접 만든 것들.

마케터가 필요한 도구를 직접 코드로 만들어 운영을 자동화·시스템화한 프로젝트입니다.

★ 대표 프로젝트 · 바이브 코딩

AI CMO — 퍼포먼스 마케팅 통합 분석·진단 대시보드

Python · Streamlit · Plotly · Google Ads/Meta/TikTok API · SQLite · 2026.01 ~ 현재
한 줄 소개

구글애즈·메타·틱톡을 매번 따로 접속하지 않고 한 화면에서 통합 진단하는, 마케터가 직접 코딩한 분석 도구. 기성 툴에 없는 진단(중복DB율·캠페인 수명·실전환 CPA)을 직접 지표로 설계해 넣었습니다.

핵심 기능
  • 📊 실시간 통합 조회 — 3채널(구글·메타·틱톡) 비용·전환·CPA·CTR·CVR을 한 화면에서
  • 🔍 단계별 원인 추적 — 캠페인 → 광고그룹 → 소재로 한 단계씩 좁혀 원인 분해
  • 📊 코호트 수명 진단 — 캠페인을 '투입 후 경과 주차'로 정렬해 풀 소진 시점 포착
  • 🧪 A/B 테스트 + 평가 루프 · 💡 인사이트 축적(반복 가능한 의사결정)
  • 🤖 자동 모니터링 봇 — CPA 급등·과지출 이상 신호를 텔레그램 알림
직접 설계한 지표 (차별점)

중복DB율(풀 소진 선행 신호) · 실전환 기준 CPA(API 누락 보정) · 캠페인 수명 코호트 · CTR-CVR-CPM 3축 분류 — 기성 지표로 안 보이는 문제를 잡으려 직접 만든 지표들입니다.
